# Sockets

Demo code for implementing Event Broadcasting with Laravel 5.1 and Sockets.io, based on the Taylor Otwell's video.

## Technologies:

- Laravel 5.1
- NodeJS
- Socket.io
- VueJS

## Concepts:

- Events
- Event broadcasting
- Web sockets
